Built With Claude

The Anthropic workflow — from architecture to deployment.

Every site I build uses Claude as a pair-programming partner. Not copy-paste from ChatGPT — a genuine collaborative workflow where the AI understands the full architecture and makes informed decisions alongside me.

01

Architecture

Claude helps design the system before any code is written. We discuss trade-offs between SSR and SSG, debate database schemas, map out API routes. The AI sees the full picture — not just the file I'm editing. Every architectural decision is reasoned through, not guessed at.

02

Pair Programming

Real-time collaboration. I describe what I need, Claude writes the implementation. I review, adjust, push back. It's not about generating code — it's about having a second brain that remembers every dependency, every edge case, every decision we made three hours ago.

03

Chatbot Engineering

Training AI personas is its own discipline. System prompts are 2,000+ tokens of carefully structured personality, knowledge boundaries, anti-hallucination rules, and voice guidelines. Claude helps write prompts that make another AI behave correctly. It's AI training AI.

04

Deployment

From local development to global edge deployment. Claude handles Wrangler configs, R2 uploads, D1 migrations, DNS records, secret management. The entire infrastructure is code-defined and Claude understands every layer — from the Worker fetch handler to the Turnstile verification flow.

Every site is a template. Swap the logo. Change the palette. Feed it your facts. It becomes yours.
